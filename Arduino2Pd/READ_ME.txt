/*---------------------------------------------Arduino2Pd v0.1---------------------------------------------------*/

"Arduino2Pd.ino" sketch is set up to work with Pure Data patch "ArduinoSerial.pd". in the folder there is a file called "Arduino2Pd.pd" that show you the abstractions made to work with this Arduino sketch ("ArduinoSerial.pd" and "ArduinoScaler.pd" - the latter is nested into the first)
